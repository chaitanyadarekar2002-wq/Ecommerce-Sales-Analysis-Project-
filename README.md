# 🛒 Ecommerce Sales Analysis Project

## 📌 Project Overview

This project focuses on analyzing ecommerce sales data to generate meaningful insights about **sales performance, customer behavior, and regional trends**.

The goal is to use **SQL and data visualization tools** to answer key business questions and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze total revenue and sales trends over time
* Understand product and category performance
* Identify top customers and buying patterns
* Evaluate regional sales distribution
* Calculate key metrics like **AOV (Average Order Value)**

---

## 🗂️ Database Schema

The project is based on 5 main tables:

1. **Customers** – Customer details
2. **Products** – Product information
3. **Orders** – Order-level data
4. **OrderDetails** – Transaction-level details
5. **Regions** – Geographic information

---

## ❓ Key Business Questions

### 1. General Sales Insights

* Total revenue generated
* Revenue excluding returns
* Monthly and yearly revenue trends
* Revenue by product/category
* Average Order Value (AOV)
* AOV by month/year
* Average order size by region

### 2. Customer Insights

* Top 10 customers by revenue
* Repeat customer rate
* Average time between orders
* Customer segmentation based on spending

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data extraction and analysis
* **Tableau / Power BI** – Data visualization
* **Excel / CSV** – Dataset handling
* **GitHub** – Project version control

---

## 📊 Key Metrics Explained

* **Revenue** = Quantity × Price
* **AOV (Average Order Value)** = Total Revenue / Number of Orders
* **Repeat Customer Rate** = Returning Customers / Total Customers
